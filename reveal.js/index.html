Rust in a nutshell

  • compiled to native
  • functional (kind of, also procedural, imperative)
  • concurrent
  • open source !
  • memory safety (thanks lifetimes)
  • integrated package manager (thanks Cargo)
  • awesome compiler
  • toolchains for lots of platforms (here)
Rust is multiparadigm in general, but focuses on functional and concurrent paradigms. Cargo using crates. Rustup to install rust

Rust in a crapshell

  • very steep learning curve
  • functional with borrows - problems with cyclic data structures (like graphs)
  • quite long compilation time
  • no official IDE
  • tempting 'unsafe rust'
  • no enterprise level libraries

Rust in.. general

  • no destructors
  • no objectivity (for some at least)
    • method overloading -> trait boundaries
    • little different polymoprhysm (have a read)
    • inheritance -> composition
    • interface -> trait
    • class -> struct
    • no constructors
  • very restrictive
